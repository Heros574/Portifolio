Olá! Meu nome é Heros e sou um desenvolvedor full stack em formação, apaixonado por tecnologia, design e soluções criativas. Atualmente, estudo no Instituto Federal de Minas Gerais – Campus Ribeirão das Neves, onde curso o Técnico em Informática integrado ao Ensino Médio. Durante minha jornada no IFMG, venho adquirindo conhecimentos sólidos em desenvolvimento web, programação, bancos de dados e design de interfaces, sempre buscando unir a lógica da programação com uma boa experiência visual para o usuário. Tenho experiência acadêmica com tecnologias como HTML, CSS, JavaScript, Bootstrap, PHP e também com ferramentas de design gráfico.

Rede de interação de Usuarios

Basicamente esse projeto é uma plataforma web inspirada no Reddit, desenvolvida para demonstrar minhas habilidades em programação e design. O sistema permite interação entre usuários por meio de postagens e comentários. Utilizei PHP com banco de dados no back-end e Bootstrap + jQuery no front-end para criar uma interface responsiva, dinâmica e funcional.
